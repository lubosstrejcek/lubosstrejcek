## Featured Project

### Victron Energy Monitoring Stack

Docker-based monitoring solution for Victron Energy systems, created during **Victron Energy Training Center Brno - Node-RED Training**.

| Repository | Description |
|------------|-------------|
| [victron-nodered](https://github.com/lubosstrejcek/victron-nodered) | Node-RED flows for MQTT data collection |
| [victron-influxdb](https://github.com/lubosstrejcek/victron-influxdb) | InfluxDB time-series database |
| [victron-grafana](https://github.com/lubosstrejcek/victron-grafana) | Grafana dashboards for visualization |

**Stack:** Node-RED + InfluxDB 2.x + Grafana

```
Ekrano GX (MQTT) → Node-RED → InfluxDB → Grafana
```

Feel free to use these repositories for inspiration or as a starting point for your own Victron monitoring setup.
